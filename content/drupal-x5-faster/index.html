Drupal, only x5 faster
09 Jan 2014
|
By Amitai Burstein
  • Zariz
  • Drupal-planet

This is the live branch on the Drupal site

This is the static site created out of the live site. It looks the same, but loads x5 faster.

The static site was created using the Zariz generator, and executing grunt stage. For scalability reasons, the grunt task asks from the Drupal site only for the incremental changes by calling /zariz-pages. The output is a JSON with the pages that need to be grabbed.

{
  insert: [
    "node/2",
    "node/3",
    "node/4",
    "node/5",
    "index.php"
  ],
  delete: [ ]
}

We create a new branch and add new content ...

... and merge it back to the live branch.

The output of /zariz-pages reflects the changes

{
  insert: [
    "node/7",
    "index.php"
  ],
  delete: [ ]
}

The static site was updated.
